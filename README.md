## 소개

파티원구함은 단체 약속 시간 확정을 도와주는 웹 서비스에요. 인원수, 날짜, 시간을 정해 약속 페이지를 만들고, 링크를 공유해 인원수가 채워지면 이메일로 알림을 받을 수 있어요.
